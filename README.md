### 🎮 23Tile Puzzle Game

Check out my [23Tile Sliding Puzzle Game](23tile.html)! It's a fun brain teaser where you need to arrange tiles numbered 1-23 in order by sliding them into the empty space.

**Features:**
- 🧩 Classic sliding puzzle with 23 tiles in a 4x6 grid
- 🔀 Shuffle button to randomize the board
- 🔄 Reset button to return to the initial state
- 📊 Move counter to track your progress
- 🏆 Win detection with celebration message

**How to Play:**
1. Open `23tile.html` in your browser
2. Click "Shuffle" to start a new game
3. Click on tiles adjacent to the empty space to move them
4. Arrange all tiles in order from 1 to 23
5. Try to solve it in as few moves as possible!
